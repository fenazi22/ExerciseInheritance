# ExerciseInheritance

A Java practice repository focused on inheritance in object-oriented programming.

## Project Overview

This repository is used to practice how classes can inherit shared attributes and behavior from parent classes.

## Main Topics

- Java fundamentals
- Object-Oriented Programming
- Inheritance
- Parent classes
- Child classes
- Method overriding
- Code reuse

## Learning Goals

- Understand inheritance relationships.
- Reduce repeated code by using parent classes.
- Practice extending classes.
- Apply method overriding when needed.
- Build cleaner Java class structures.
